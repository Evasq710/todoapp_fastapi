pip install 'fastapi[standard]'
> The key libraries included in the "standard" extra are:
    - Uvicorn: An ASGI server implementation for running the FastAPI app
    - Pydantic: A data validation and settings management library. "email-validator" (a dependency of Pydantic) is also included
    - python-multipart: Required for parsing form data, including file uploads
    - Jinja2: A popular templating engine for rendering HTML templates
    - python-jose: A library for handling JSON Object Signing and Encryption (JOSE), often used for JWTs (see pyjwt[crypt])
    - passlib: Provides secure password hashing functionality
    - SQLAlchemy: A powerful Object-Relational Mapper (ORM) for interacting with databases.
    - asyncpg: An asynchronous PostgreSQL driver for use with SQLAlchemy
    - databases: A library providing asynchronous database support, often used with SQLAlchemy
    - fastapi-cloud-cli: A commnand-line interface for deploying FastAPI applications to FastAPI Cloud

# Database migration tool & Database enhancement tool
pip install alembic
    - alembic init <folder_name>: Initializes a new, generic environment. Generally: folder_name=alembic
    - alembic revision -m <message>: Creates a new revision of the environment
    - alembic upgrade <revision_id>: Run our upgrade migration to our database
    - alembic downgrade -1: Run our downgrade migration to our database (Revert a migration)
